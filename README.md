# 多人聊天室-my-ATP-chatroom

支付宝体验科技前端项目，在蚂蚁的工程师指导下完成

使用nodejs，需要安装koa,socket等包
可以运行在本地服务器上，命令：node server.js，
端口设置的3000，本地访问：http://127.0.0.1:3000

安装localtunnel包可以映射到外网，命令：lt --port 3000
